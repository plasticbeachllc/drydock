# macOS Terminal QOL Setup Runbook

> Run each section in order on your Mac. Copy-paste the commands into your current Terminal.app — by the end, you'll switch to Ghostty and never look back.

---

## Prerequisites

Make sure Homebrew is installed:

```bash
# Skip if you already have brew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

---

## 1. Terminal Emulator — Ghostty

```bash
brew install --cask ghostty
```

We'll configure Ghostty after installing everything else (fonts, etc).

---

## 2. Font — Maple Mono (with Nerd Font icons built in)

```bash
brew install --cask font-maple-mono-nf
```

```bash
fc-list | grep -i "Maple"
```

You should see entries like `Maple Mono NF`, `Maple Mono NF Bold`, etc.

## 7. Configure Sheldon (Zsh Plugin Manager)

Create the sheldon config:

```bash
mkdir -p ~/.config/sheldon
```

```bash
cat > ~/.config/sheldon/plugins.toml << 'EOF'
shell = "zsh"

# --- Completions (load early) ---
[plugins.zsh-completions]
github = "zsh-users/zsh-completions"

# --- Core plugins ---
[plugins.zsh-autosuggestions]
github = "zsh-users/zsh-autosuggestions"

[plugins.zsh-syntax-highlighting]
github = "zsh-users/zsh-syntax-highlighting"

[plugins.zsh-history-substring-search]
github = "zsh-users/zsh-history-substring-search"

# --- fzf-tab (replaces default tab completion with fzf) ---
[plugins.fzf-tab]
github = "Aloxaf/fzf-tab"
EOF
```

Lock the plugins (downloads them):

```bash
sheldon lock
```

## 8. Configure Starship Prompt

```bash
mkdir -p ~/.config
```

```bash
cat > ~/.config/starship.toml << 'EOF'
# Minimal, fast prompt with git + directory + jj support
format = """
$directory\
$git_branch\
$git_status\
$character"""

[directory]
truncation_length = 3
truncate_to_repo = true

[git_branch]
format = "[$branch]($style) "
style = "bold purple"

[git_status]
format = '([$all_status$ahead_behind]($style) )'
style = "bold red"

[character]
success_symbol = "[❯](bold green)"
error_symbol = "[❯](bold red)"
EOF
```

## 9. Configure Ghostty

```bash
mkdir -p ~/.config/ghostty
```

```bash
cat > ~/.config/ghostty/config << 'EOF'
# Font
font-family = "Maple Mono NF"
font-size = 14

# Theme — start with a clean dark theme
theme = catppuccin-mocha

# Window
window-padding-x = 12
window-padding-y = 8
window-decoration = true
macos-titlebar-style = hidden

# Behavior
copy-on-select = clipboard
mouse-hide-while-typing = true
confirm-close-surface = false
shell-integration = zsh

# Keybindings for splits
keybind = cmd+d=new_split:right
keybind = cmd+shift+d=new_split:down
keybind = cmd+shift+enter=toggle_split_zoom
keybind = cmd+alt+left=goto_split:left
keybind = cmd+alt+right=goto_split:right
keybind = cmd+alt+up=goto_split:top
keybind = cmd+alt+down=goto_split:bottom

# Scrollback
scrollback-limit = 10000
EOF
```

## 10. Configure .zshrc

Back up your existing .zshrc first:

```bash
[ -f ~/.zshrc ] && cp ~/.zshrc ~/.zshrc.backup.$(date +%Y%m%d)
```

Now write the new config:

```bash
cat > ~/.zshrc << 'ZSHRC'
# ============================================================
# Taylor's ZSH Config — Generated by terminal QOL setup
# ============================================================

# --- Sheldon (plugin manager) ---
eval "$(sheldon source)"

# --- Starship prompt ---
eval "$(starship init zsh)"

# --- Zoxide (smart cd) ---
eval "$(zoxide init zsh)"

# --- direnv ---
eval "$(direnv hook zsh)"

# --- fzf ---
[ -f ~/.fzf.zsh ] && source ~/.fzf.zsh

# --- fzf config ---
export FZF_DEFAULT_COMMAND='fd --type f --hidden --follow --exclude .git'
export FZF_CTRL_T_COMMAND="$FZF_DEFAULT_COMMAND"
export FZF_ALT_C_COMMAND='fd --type d --hidden --follow --exclude .git'
export FZF_DEFAULT_OPTS='
  --height 40%
  --layout=reverse
  --border
  --color=bg+:#313244,bg:#1e1e2e,spinner:#f5e0dc,hl:#f38ba8
  --color=fg:#cdd6f4,header:#f38ba8,info:#cba6f7,pointer:#f5e0dc
  --color=marker:#b4befe,fg+:#cdd6f4,prompt:#cba6f7,hl+:#f38ba8
'

# --- History config ---
HISTSIZE=50000
SAVEHIST=50000
HISTFILE=~/.zsh_history
setopt EXTENDED_HISTORY       # Write timestamps to history
setopt HIST_EXPIRE_DUPS_FIRST # Expire duplicates first
setopt HIST_IGNORE_DUPS       # Don't record immediate duplicates
setopt HIST_IGNORE_SPACE      # Don't record commands starting with space
setopt HIST_VERIFY            # Show command before executing from history
setopt SHARE_HISTORY          # Share history between sessions
setopt INC_APPEND_HISTORY     # Write to history immediately

# --- Key bindings ---
# History substring search (up/down arrow)
bindkey '^[[A' history-substring-search-up
bindkey '^[[B' history-substring-search-down

# --- Aliases: CLI tool replacements ---
alias ls='eza --icons --group-directories-first'
alias ll='eza --icons --group-directories-first -la'
alias lt='eza --icons --tree --level=2'
alias cat='bat --paging=never'
alias grep='rg'
alias find='fd'
alias du='dust'
alias top='btop'
alias curl='xh'
alias sed='sd'
alias diff='delta'
alias lg='lazygit'
alias lj='lazyjj'

# --- Git aliases (for repos not yet on jj) ---
alias gs='git status'
alias gd='git diff'
alias gl='git log --oneline -20'
alias gp='git push'
alias gpf='git push --force-with-lease'

# --- jj aliases ---
alias js='jj status'
alias jl='jj log'
alias jd='jj diff'
alias jn='jj new'
alias jc='jj commit'
alias jds='jj describe'

# --- delta as git pager ---
export DELTA_PAGER="less -R"

# --- bat theme ---
export BAT_THEME="Catppuccin Mocha"

# --- PATH additions ---
# Cargo (for Rust-installed tools)
[ -d "$HOME/.cargo/bin" ] && export PATH="$HOME/.cargo/bin:$PATH"

# jj-fzf (if cloned manually)
[ -d "$HOME/.jj-fzf" ] && export PATH="$HOME/.jj-fzf:$PATH"

ZSHRC
```

## 11. Configure Git to use Delta

```bash
git config --global core.pager delta
git config --global interactive.diffFilter "delta --color-only"
git config --global delta.navigate true
git config --global delta.dark true
git config --global delta.syntax-theme "Catppuccin Mocha"
git config --global delta.side-by-side true
git config --global delta.line-numbers true
git config --global merge.conflictstyle diff3
git config --global diff.colorMoved default
```

## 12. Configure jj

```bash
mkdir -p ~/.config/jj
```

```bash
cat > ~/.config/jj/config.toml << 'EOF'
[user]
name = "Taylor"
email = "YOUR_EMAIL_HERE"

[ui]
default-command = "log"
diff.format = "git"
pager = "delta"

[git]
auto-local-bookmark = false
EOF
```

> **Important:** Replace `YOUR_EMAIL_HERE` with your actual email.

## 14. Quick Test Drive

Try these out to feel the difference:

```bash
# eza: beautiful file listing
ls ~/worktable

# eza: tree view of a project
lt ~/worktable/keel

# bat: syntax-highlighted file viewing
bat ~/.zshrc

# ripgrep: fast search
rg "TODO" ~/worktable --type ts

# fd: find files
fd "\.ts$" ~/worktable/keel

# zoxide: jump to directories (builds up over time)
z keel       # after visiting ~/worktable/keel once

# fzf: fuzzy find (Ctrl-R for history, Ctrl-T for files)
# Try pressing Ctrl-R now!

# delta: pretty git diffs
cd ~/worktable/keel && git log --oneline -5

# lazygit: TUI git client
cd ~/worktable/keel && lg

# jj: try it on an existing repo
cd ~/worktable/keel && jj git init --colocate
jj log
jj status

# lazyjj: TUI for jj
lj
```

## Ghostty Keybindings Cheat Sheet

| Action | Keybinding |
|--------|------------|
| Split right | `Cmd+D` |
| Split down | `Cmd+Shift+D` |
| Navigate splits | `Cmd+Alt+Arrow` |
| Zoom split | `Cmd+Shift+Enter` |
| New tab | `Cmd+T` |
| Close tab/split | `Cmd+W` |

## What's Next

- **Tune Starship**: Edit `~/.config/starship.toml` to show more/less info
- **Tune Ghostty theme**: Try `catppuccin-latte` for light mode, or `tokyo-night` for something different
- **Learn jj**: Start with Steve Klabnik's tutorial at https://steveklabnik.github.io/jujutsu-tutorial/
- **tmux**: When you need persistent sessions on your Hetzner box, we can layer tmux on top
- **Ghostty splits vs tmux**: Use Ghostty splits for local work, tmux only for remote sessions
